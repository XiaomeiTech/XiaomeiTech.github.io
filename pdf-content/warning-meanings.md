## 警告标识说明

本文档中使用了以下安全警告标识，用于提醒用户在操作和维护设备时注意潜在风险：


::: danger
表示即将发生的危险情况，如果不避免将导致死亡或严重伤害
:::

::: warning
表示潜在的危险情况，如果不避免可能导致死亡或严重伤害
:::

::: caution
表示潜在的危险情况，如果不避免可能导致轻微或中度伤害
:::

::: notice
表示与人身伤害无关的操作注意事项，如设备损坏风险
:::

::: info
表示补充信息，提供额外的背景或说明
:::



**安全操作要求**

1. 在安装、操作或维护设备之前，请仔细阅读相关技术文档
2. 只有经过培训的合格人员才能进行设备的安装和维护
3. 务必遵守所有相关的安全法规和标准
4. 在进行任何电气连接之前，确保电源已断开
5. 使用适当的个人防护装备（PPE）
6. 如发现设备异常或损坏，立即停止使用并联系技术支持
