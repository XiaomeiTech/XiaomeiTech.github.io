# PDF 手册下载

本页面提供各产品线 PDF 文档的下载入口。所有 PDF 由 GitHub Actions 自动编译生成，确保内容与线上文档同步。

## 下载方式

- **最新版**：前往 [GitHub Releases](https://github.com/XiaomeiTech/XiaomeiTech.github.io/releases/tag/latest-pdf) 下载最新的 PDF 文件
- **历史版本**：在 [Releases 页面](https://github.com/XiaomeiTech/XiaomeiTech.github.io/releases) 查看所有历史版本

## 文档类型

| 类型 | 说明 | 文件名 |
|------|------|--------|
| 数据手册 | 产品规格与技术参数概览 | `datasheet.pdf` |
| 应用指南 | 开发环境搭建与使用指引 | `app-guide.pdf` |
| 单品手册 | 单个产品的详细操作手册 | `product-manual.pdf` |
| 系列合并手册 | 全系列产品文档汇总 | `series-manual.pdf` |
