# 治具无线负压检测（低速线）
在电池 Pack 电芯真空上料环节，常因密封件老化、气源波动等导致负压不稳，易造成吸附力下降、电芯飞脱，产生物料损耗和停机风险。为此，小美技术开发治具无线负压检测方案，实时监控气压临界报警，从源头保障吸附稳固与产线安全。

目前该项目已在某锂电制造厂商的“低速线”验证并通过测试，本项目针对常规载具实现真空度监测智能化升级，通过改造原治具、加装无线真空传感器、通讯基站与无线充电模块，实现真空度实时监测、数据无线回传与自动无线充电。真空检测传感器精准识别载具真空值并将数据稳定上传至 PLC，当真空度低于设定值时报警。该项目有效提升客户产线运行稳定性与参数可视性。改造后设备运行可靠、故障率低，为客户生产安全提供有力保障场景，有效提升产线运行稳定性与参数可视性，为客户高效安全生产提供坚实支撑。

### 改造后实现的功能
- **改造常规载具底板，增加真空度监测传感器，实现真空度监测，无线数据回传。**
- **增加无线通讯基站，接收处理传感器回传的数据，通过ModbusTCP回传给PLC。**
- **增加无线充电模组，实现在工位停靠补真空时为电池进行充电。**
