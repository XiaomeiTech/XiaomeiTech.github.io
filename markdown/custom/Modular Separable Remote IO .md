# 模块可分离式远程IO
针对电池柜项目中大规模灯珠与光电传感器的IO控制需求，我们为客户定制了模块可分离式远程I/O解决方案，采用“一拖六分布式架构”，将定制灯组传感器IO板与EtherCAT通讯接口模块分离设计，既实现了3×6路数字输出、1×6路数字输入的现场信号采集与控制，又通过背板总线支持64节点联动扩展，大幅简化了传统点对点布线的现场接线工作，显著降低了线缆与施工成本，同时方案支持快速更换与功能迭代，兼容主流PLC系统，为客户提供了一套高可靠、易维护、可扩展的低成本IO控制方案。

## 定制灯组传感器IO板 EM7230L

### 应用
为客户定制灯组传感器IO板，采用 1 拖 6 分布式设计，实现传感器 IO 模块与主控单元的独立拆装与灵活部署，适配多节点灯组传感场景的模块化扩展需求，支持现场快速更换与功能迭代

### 规格介绍
- **工作电压为 DC 24V，适配宽压范围 DC 20~28V**
- **背板总线支持 64 节点联动扩展**
- **定制化设计 3×6 路数字量输出通道**
- **定制化设计1×6 路数字量输入通道**  

## 定制EC接口模块  IM2610C

### 应用
为用户定制EC接口模块，作为可分离式远程 IO 系统的通讯接口核心单元，与定制灯组传感器 IO 板配套使用，承担分布式 IO 模块与上位 PLC 之间的高速数据交互与协议转换功能

### 规格介绍
- **工作电压为 DC 24V，适配宽压范围 DC 20~28V**  
-**背板总线支持 64 节点联动扩展** 
- **支持 EtherCAT 总线协议串联组网，兼容欧姆龙等PLC**
